# 一、位运算

> 由于位运算在实际开发过程中几乎不会使用到，因此只记录一些特殊的快速计算方法

## 1.快速计算2的倍数问题

```python
n << 1 -> 计算 n*2
n >> 1 -> 计算 n/2，负奇数的运算不可用
n << m -> 计算 n*(2^m)，即乘以 2 的 m 次方
n >> m -> 计算 n/(2^m)，即除以 2 的 m 次方
1 << n -> 2^n
```

## 2.快速交换两个整数

```python
a ^= b
b ^= a
a ^= b
```
